# Digital Diary

## Project Title
Digital Diary - Java Notes Manager

## Project Description
Digital Diary is a Java-based desktop application developed using Java Swing. The application allows users to securely log in and manage personal notes digitally. Users can create, save, open, and clear notes through a simple graphical user interface. Notes are stored using Java File Handling, making the application lightweight and easy to use.

This project demonstrates the implementation of Object-Oriented Programming concepts, GUI development, event handling, exception handling, and file handling in Java.

## Features

- User Login Authentication
- Create Personal Notes
- Save Notes to a Text File
- Open Previously Saved Notes
- Clear Notes
- User-Friendly Graphical Interface
- File Handling using Java
- Exception Handling for Better Reliability

## Technologies Used
